## Sobre mim

Sou o Gustavo Zago, desenvolvedor full stack com experiência na construção de interfaces modernas, backends escaláveis e produtos digitais.

No meu trabalho, gosto de unir experiência visual, organização de produto e implementação prática, transitando entre frontend e backend com foco em soluções reais para e-commerce, CMS e aplicações web.

Atualmente, meu perfil público no GitHub destaca minha base em Computer Engineering e projetos que passam por frontend, backend e experimentos práticos.
